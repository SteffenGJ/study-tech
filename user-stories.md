# User stories

## Adding flashcards

- As a user, I want to be able to add a new flashcard
- - As a user adding a flashcard, I would like certainty to whether the new flashcard was added
- - As a user adding a flashcard, I want to know if my new flashcard for some reason wasn't saved
- - As a user adding a flashcard, I don't want to add the info on the same flashcard once again if some kind of error occurs
- - As a user adding a flashcard, I want to be able to add an image as an answer (and in the question)
- - As a user adding a flashcard, I don't want to add the same subject- and topic-specific info over and over again

## Practicing with flashcards

- As a user, I want to be able to practice with available flashcards
- - As a user practicing with flashcards, I want to be presented with the question and have the ability to reveal the correct answer on the flashcard
- - As a user practicing with flashcards, I want to be able to select a specific subject and topic to practice flashcards from
- - As a user practicing with flashcards, I want to be able to practice flashcards from multiple subjects and topics of my choice at once
- - As a user practicing with flashcards, I want to be able to choose whether the order of the flashcards I practice is random or by date, topic etc.
- - As a user practicing with flashcards, I want to be able to mark flashcards by how certain I am that I know the correct answer
- - As a user practicing with flashcards, I want to be able to practice flashcards by certainty in my answer, e.g. practicing the ones I have least degree of certainty in
- - As a user practicing with flashcards, I want to be able to practice flashcards from certain subjects and topics and select one or multiple certainty degrees to practice from

## Editing flashcards

- As a user, I want to be able to edit an existing flashcard
- - As a user editing a flashcard, I want to be able to change the subject, topic, question and answer
- - As a user editing a flashcard, I want certainty to whether my edit was succesful
